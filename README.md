## Instruction to use: 

### download it
``` 
git clone https://github.com/OleksiiCherevan/OleksiiCherevan.github.io
```


### select directory
```
cd OleksiiCherevan.github.io
```
### install dependencies (yarn or npm)
* if you do not have yarn:
``` 
npm add -g yarn
```
* install dependencies via yarn
```
yarn
```

### run app
```
yarn start
```
